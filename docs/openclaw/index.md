# 🦞 OpenClaw

> 打造你的24小时AI助手团队

OpenClaw 是一个开源的 AI 助手框架，让你可以在本地运行属于自己的 AI 助手。

---

## 什么是 OpenClaw？

OpenClaw 让你能够：

- 🤖 **运行本地 AI 助手** - 数据完全私有
- 💬 **连接多平台** - Telegram、Discord、飞书等
- 🧩 **扩展技能** - 通过 Skills 扩展能力
- 👥 **多 Agent 协作** - 组建 AI 团队
- ⏰ **定时任务** - 7×24小时运转
