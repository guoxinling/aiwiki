# 用AI整理学习笔记

> 🎯 **场景**：看完一篇文章/视频，想整理成自己的笔记
> 
> 💡 **解决方案**：AI帮你提炼要点，生成结构化笔记

---

## 快速整理流程

### 第一步：收集原始内容

把需要整理的内容提供给AI：
- 文章链接（AI可以读取部分网页）
- 复制粘贴的文本
- 视频转录的文字稿

---

### 第二步：用AI提炼核心

**提示词模板1：文章总结**
```
请帮我总结以下文章的核心内容：

[粘贴文章内容]

要求：
1. 用3句话概括核心观点
2. 列出5个关键要点
3. 提取3个对我有用的行动建议
4. 格式清晰，适合收藏复习
```

**提示词模板2：结构化笔记**
```
请将以下内容整理成结构化的学习笔记：

[粘贴内容]

格式要求：
【核心概念】一句话定义
【关键要点】用 bullet points 列出
【思维导图结构】用层级缩进表示逻辑关系
【我的思考】留3个空白让我自己填
【行动清单】具体可执行的行动
```

---

### 第三步：深度加工

**方法1：生成类比**
```
用类比的方式解释这个概念，让完全不懂的人也能听懂。
```

**方法2：制作记忆卡片**
```
将以上内容转换成5张记忆卡片（Q&A形式），
正面是问题，背面是答案，适合用Anki复习。
```

**方法3：绘制关系图**
```
用文本描述这些概念之间的关系，
可以用缩进或箭头表示层级和关联。
```

## 实战示例

### 输入：一段关于"复利"的文章

```
复利是指利息也计入本金产生新的利息，俗称"利滚利"。
爱因斯坦曾说复利是世界第八大奇迹。
假设你每年投资1万元，年化收益率10%，
30年后你将拥有约180万元，而本金只有30万元。
关键在于时间和坚持。
```

### AI整理后的笔记

```
📚 复利 - 学习笔记

【一句话定义】
复利 = 利息再生利息，利滚利

【核心公式】
最终金额 = 本金 × (1 + 利率)^时间

【关键要点】
• 复利的力量来自时间，不是本金多少
• 时间越长，增长越惊人（指数增长）
• 坚持比聪明更重要

【实际例子】
每年存1万，年收益10%，30年 = 180万
本金30万，增值150万（5倍！）

【类比理解】
就像滚雪球，越滚越大，关键是：
1. 有雪（本金）
2. 有坡（时间）
3. 一直滚（坚持）

【应用场景】
□ 理财投资
□ 技能学习（每天进步1%）
□ 习惯养成

【我的行动】
□ 开始定期定投
□ 每天学习30分钟，坚持一年
□ _____________
```

## 不同类型内容的整理方法

### 视频课程

```
视频主题：[课程名称]

请帮我：
1. 列出课程大纲结构
2. 每节课的核心要点
3. 重要的概念定义
4. 实操步骤（如有）
5. 推荐复习的重点章节
```

### 书籍阅读

```
书名：《书名》
作者：xxx

请帮我整理：
【一句话总结】这本书讲什么
【核心观点】3-5个主要论点
【对我有用的内容】结合我的工作/生活
【行动计划】读完后要做什么
【延伸阅读】类似主题的其他书
```

### 会议/讲座

```
会议主题：xxx
时间：xxx

请帮我整理成会议笔记：
【会议目标】这次会议要解决什么问题
【关键决策】做了哪些决定
【行动项】谁、做什么、什么时候完成
【我的任务】我需要做什么
【疑问待澄清】还需要了解什么
```

## 笔记模板库

### 康奈尔笔记法（AI版）

```
请用康奈尔笔记法整理以下内容：

[你的内容]

格式：
┌──────────┬─────────────────────┐
│ 线索/问题 │     笔记内容         │
│  (左侧)   │     (右侧)           │
├──────────┴─────────────────────┤
│          总结 (底部)            │
└────────────────────────────────┘
```

### PARA笔记法

```
请用PARA方法分类整理这些信息：

P - Projects（项目）
A - Areas（领域）
R - Resources（资源）
A - Archives（归档）

[你的内容]
```

## 效率提升技巧

### 1. 建立笔记SOP

每次学习都按这个流程：
1. 快速浏览，标记不懂的地方
2. AI辅助整理核心笔记
3. 添加自己的思考和批注
4. 设置复习提醒
5. 定期回顾和链接

### 2. 标签系统

给笔记打标签，方便查找：
- #待深入学习
- #已完成
- #可分享
- #有实践价值

### 3. 定期回顾

```
请帮我回顾以下笔记，提取还需要我关注的要点：

[粘贴之前的笔记]

重点关注：
1. 我还没有实践的行动项
2. 理解还不够深入的概念
3. 可以应用到当前工作的内容
```

---

## ⚠️ 注意事项

1. **AI是辅助，不是替代** - 一定要加入自己的思考
2. **核实关键信息** - 特别是数据、人名、公式
3. **保持简洁** - 笔记是为了复习，不是复制原文
4. **及时复习** - 整理的笔记要定期回顾才有价值
