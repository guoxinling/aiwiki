## 实际案例

### 案例1：用Notion AI写会议纪要

**输入**：
```
总结这次会议的要点：
[粘贴会议录音转文字]
```

**输出**：
Notion AI自动生成：
- 会议决议
- 行动项（带负责人）
- 时间节点

---

### 案例2：ChatExcel处理数据

**输入**：
```
"把A列和B列的数据求和"
"找出销售额最高的前10名"
"生成销售额趋势图"
```

**效果**：
不用写公式，说话就能处理Excel

---

### 案例3：Gamma做PPT

**步骤**：
1. 输入主题："远程办公效率提升方案"
2. 选择模板风格
3. AI自动生成完整PPT
4. 微调内容和数据

**耗时**：从3小时缩短到30分钟
