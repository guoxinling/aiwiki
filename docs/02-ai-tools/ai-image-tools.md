## 快速上手：即梦AI

### 1. 访问官网
jimeng.jianying.com

### 2. 输入描述
```
一只可爱的橘猫，在阳光下打盹，
写实风格，高清8K
```

### 3. 选择风格
- 写实照片
- 动漫卡通
- 油画水彩
- 中国风

### 4. 生成下载
点击生成，选择满意的图片下载

## 提示词技巧

**公式**：主体 + 细节 + 风格 + 质量

**示例**：
```
主体：未来城市夜景
细节：霓虹灯光，飞行汽车，雨夜街道
风格：赛博朋克，电影感
质量：8K高清，电影级渲染
```
