用AI写工作周报

🎯 场景：每周五都要写周报，总是不知道写什么

💡 解决方案：用AI帮你整理和润色

准备工作​

收集本周工作内容​

先列出你这周做过的事情（简单记法即可）：

周一：和客户A开会，讨论了产品需求
周二：写项目方案，修改了合同
周三：处理了3个客户投诉
周四：参加部门培训
周五：整理文档，准备下周会议

使用AI生成周报​

提示词模板​

帮我将以下内容整理成一份正式的工作周报，要求：
1. 结构清晰，分工作完成情况、下周计划两部分
2. 语言专业、简洁
3. 突出重点成果

我的工作记录：
[粘贴你的工作内容]

示例​

输入：

帮我将以下内容整理成工作周报：
- 周一：完成产品需求文档编写
- 周二：和技术部对接开发排期
- 周三：处理了5个客户反馈的问题
- 周四：完成季度数据分析报告
- 周五：组织团队周会

AI输出：

## 本周工作总结

### 一、工作完成情况

1. **产品需求文档编写**
   - 完成XX产品V2.0版本需求文档
   - 已提交产品部审核

2. **项目协调推进**
   - 与技术部对接开发排期，确定开发周期
   - 明确各阶段交付时间节点

3. **客户支持**
   - 处理客户反馈问题5个，解决率100%
   - 收集产品优化建议3条

4. **数据分析**
   - 完成Q3季度数据分析报告
   - 提出改进建议2项

5. **团队管理**
   - 组织团队周会，同步项目进展

### 二、下周工作计划

- [ ] 跟进技术部开发进度
- [ ] 准备产品内测方案
- [ ] 整理客户反馈文档

进阶技巧​

1. 批量生成多份周报​

如果你有多个项目的周报要写：

我需要为以下3个项目各写一份周报，风格保持一致：

项目A：...
项目B：...
项目C：...

2. 让AI帮你提取亮点​

从以下工作内容中，提取3个最有价值的成果，用数据说话：
[你的工作记录]

3. 生成可视化图表建议​

我下周要向领导汇报，建议用什么图表展示这些数据？
[你的数据]

模板收藏​

简约版周报模板​

请用以下格式帮我写周报：

【本周完成】
1. 
2. 
3. 

【下周计划】
1. 
2. 
3. 

【需要帮助】
（如有）

工作内容：
[你的记录]

⚠️ 注意事项​

  1. 数据要准确 - AI润色可以，但数据要自己核对
  2. 别太夸张 - AI可能会过度包装，保持真实
  3. 个性化调整 - 根据公司文化和领导偏好调整语气
