💬 提示词基础

📌 概述​

提示词（Prompt） 就是你和AI对话时说的话。学会写好提示词，就像学会和同事有效沟通一样重要。

提示词的质量直接决定了AI回答的质量。同样的AI，有人能用它写出精彩的文章，有人却只能得到泛泛而谈的回答——区别就在于提示词的技巧。

🎯 为什么提示词很重要？​

1. 决定回答质量​

  • ❌ 模糊提问 → AI泛泛而谈
  • ✅ 清晰指令 → AI精准回答

2. 提高工作效率​

  • 学会提示词，3分钟完成原本30分钟的工作
  • 减少反复修改的次数

3. 释放AI潜力​

  • AI的能力边界，很大程度上由你的提示词决定
  • 好的提示词能让AI扮演专家、顾问、创意伙伴

💡 核心概念​

什么是好的提示词？​

好的提示词包含三个要素：

要素说明示例
任务让AI做什么"写一篇产品介绍"
背景相关情境信息"针对25-35岁职场人士"
要求具体格式/风格"300字以内，突出性价比"

提示词的基本结构​

【角色】你是一位...
【任务】请帮我...
【背景】情况是这样的...
【要求】需要满足以下条件...
【格式】请以...形式输出

🛠️ 实践指南​

示例1：写邮件​

❌ 不好的提示词：

"帮我写封邮件"

✅ 好的提示词：

"请帮我写一封申请调休的邮件，发给直属领导。理由是需要处理家庭紧急事务，请求批准下周一调休一天。语气正式但不生硬，200字以内。"

示例2：写文案​

❌ 不好的提示词：

"写个产品文案"

✅ 好的提示词：

"你是一位资深营销文案。请为一款降噪耳机写3条朋友圈文案，目标用户是上班族，突出'地铁上也能安静听书'的场景，每条50字以内，带emoji，风格轻松有趣。"

示例3：学习辅导​

❌ 不好的提示词：

"解释一下机器学习"

✅ 好的提示词：

"假设我是一名零基础的大学生，请用通俗易懂的语言解释什么是机器学习。请用类比的方式说明，并举2个生活中的例子。避免使用专业术语，如果必须使用，请简单解释。"

📝 写作技巧​

1. 具体而非笼统​

  • ❌ "写得好一点"
  • ✅ "使用专业术语，引用权威数据"

2. 提供背景信息​

  • ❌ "分析这个数据"
  • ✅ "这是一家奶茶店上个月的销售数据，请分析工作日和周末的销售差异"

3. 指定输出格式​

  • ❌ "总结一下"
  • ✅ "用 bullet points 总结，每条不超过20字"

4. 给出示例（Few-shot）​

请按照以下风格回复：

示例1：
问：今天天气怎么样？
答：今日晴空万里，气温适宜，正是出游好时节！

示例2：
问：这个方案可行吗？
答：此方案颇具创意，执行层面尚需斟酌。

现在请用这种风格回答：如何提高工作效率？

❓ 常见问题​

Q1: 提示词越长越好吗？

A: 不是。提示词要清晰完整，而不是冗长。关键是包含必要的信息：任务、背景、要求。通常在100-300字之间效果最好。

Q2: AI回答不满意怎么办？

A: 可以：

  1. 让AI"说得更详细/简单一些"
  2. 指出具体哪里不好，让AI修改
  3. 提供更多背景信息
  4. 换一种表达方式重新提问

Q3: 每次都要写这么长的提示词吗？

A: 不用。简单的任务一句话就够了。复杂的任务才需要详细提示词。随着使用经验增加，你会越来越熟练。

Q4: 不同AI的提示词技巧一样吗？

A: 基本原理相同，但各有特点：

  • ChatGPT：对结构化提示词响应好
  • Claude：擅长处理长文本和复杂指令
  • 文心一言：对中文语境理解更好

Q5: 怎么判断我的提示词写得好不好？

A: 看AI的回答是否：

  • ✅ 符合你的预期
  • ✅ 提供了你需要的信息
  • ✅ 格式和风格符合要求
  • ✅ 不需要大幅修改就能用

📚 进阶学习​

推荐资源​

  • 提示词框架 - 学习常用的提示词框架
  • 角色提示技巧 - 让AI扮演专家回答问题

练习建议​

  1. 从日常任务开始练习（写邮件、改文案）
  2. 对比不同提示词的效果差异
  3. 收集和整理自己常用的提示词模板

🎯 掌握提示词基础后，你会发现AI的实用性提升10倍！
